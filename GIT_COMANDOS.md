# Comandos Git para publicar os repositorios

## Backend

```bash
cd fincontrol-api
git init
git add .
git commit -m "Cria API FinControl"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/fincontrol-api.git
git push -u origin main
```

## Frontend

```bash
cd fincontrol-frontend
git init
git add .
git commit -m "Cria frontend FinControl"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/fincontrol-frontend.git
git push -u origin main
```
